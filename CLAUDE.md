# Local dev server

A dev server is already running locally on http://localhost:3000 (started manually
by the user, outside Claude Code). Do NOT start a new dev server (no preview_start,
no `next dev` / `bun`/`npm run dev` via Bash) — always reuse the one on port 3000.

For admin UI testing/review, default to opening:
http://localhost:3000/ivyjstudio/admin
